I now use AI to write a list of keywords for each class. The AI gives me the Chinese translation, then the definition and an example sentence. I then use this list as part of the lesson report which I finish writing myself. Occasionally, I ask the AI to use the list of words to create an activity or homework sheet, often without much success.
