# SDLC Analysis: Overall Summary (IT & Software Development Roles)

## Executive Summary

This analysis examines **942 interviews** across SDLC and IT-related professional roles to understand how AI is being used in Software Development Life Cycle processes.

**Note**: This is a filtered analysis focusing only on roles directly related to software development, IT, and technical work. 22 non-technical roles have been excluded.
