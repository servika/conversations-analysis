# Interview Transcripts Analysis by Professional Position

## Overview
This folder contains interview transcripts organized by the professional position/role of the interviewee. Each file contains full conversations separated by divider lines (========).

## File Format

Each position file contains:
- Full conversation transcripts
- Conversations separated by lines of equal signs (========)
- Both Assistant/AI and User messages preserved
- Original conversation flow maintained

## Usage

To read conversations for a specific profession:
1. Open the corresponding .txt file
2. Each conversation is separated by divider lines
3. Conversations include the complete interview from start to finish

## Notes

- Some conversations may be categorized as "Unknown" if the profession wasn't clearly stated or couldn't be inferred
- The categorization was done using pattern matching on self-described roles and contextual clues
- Files are named with underscores replacing spaces for compatibility
